## Фаза 1: Дизайн-аудит бота

**Запрос:** Провести полный анализ бота с точки зрения дизайна, оценить и дать вердикт.

**Что анализировал:**
- Все клавиатуры (Reply + Inline)
- User flows (регистрация, поиск, чат, after-chat, AI-персонажи)
- Копирайтинг всех текстов (locales.py)
- Система AI-персонажей
- Монетизация (Premium, подарки)

**Ключевые проблемы найденные в аудите:**
1. Тексты — технический/мануальный тон, читаются как документация, не продают
2. Клавиатура чата — кнопка "Лайк" рядом с "Жалоба" (случайные нажатия)
3. AI-персонажи — 11+ рядов кнопок для non-premium, перегруз
4. After-chat — 3 сообщения подряд (спам), можно сжать до 2
5. Профиль — данные выводятся как дамп из базы
6. Кнопки — "⚡ Поиск" непонятна, "🔍 По анкете" расплывчата
