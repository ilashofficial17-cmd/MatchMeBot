### Задача 1 — Редизайн трёх разделов locales.py

**Коммит:** `1441d61`
**Файл:** `locales.py`
**Языки:** RU / EN / ES

#### AI-меню (`ai_menu`)

**Было:**
```
🤖 ИИ чат

⚡ Энергия: X/Y

✅ Basic — 1 ⚡   ⭐ VIP — 2 ⚡   💎 VIP+ — 3 ⚡

Выбери персонажа:
```

**Стало:**
```
🤖 ИИ Чат
━━━━━━━━━━━━━━━
⚡ X / Y

Basic · 1⚡   VIP ⭐ · 2⚡   VIP+ 💎 · 3⚡

Выбери персонажа 👇
```

**Изменения:** убран лейбл «Энергия:», убран `✅` перед Basic (семантически некорректен), добавлен разделитель под заголовком, тиры сжаты в одну строку.

#### Блоки клавиатуры AI (`ai_block_*`)

**Было:** `💬 Общение`
**Стало:** `── 💬 Общение ──`

Приведено к стилю `prem_header` («── Premium ──»), который уже существовал в проекте.

#### Карточка профиля (`profile_text`)

**Было:** один разделитель, визуальная каша между интересами и статистикой.
**Стало:** второй разделитель `━━━━━━━━━━━━━━━` между блоком интересов и блоком статистики — чёткое визуальное деление.

#### Компоненты профиля

| Ключ | Было | Стало |
|------|------|-------|
| `profile_level` | `📊 Уровень: 3 — Душа компании` | `🏅 Душа компании  ·  ур. 3` |
| `profile_streak` | `🔥 Стрик: 5 дн.` | `🔥 5 дн. подряд` |
| `profile_progress` | `📈 До след. уровня: 12/20 чатов (60%)` | `   └ 12/20 · 60%` |
| `profile_progress_max` | `📈 Максимальный уровень!` | `🏆 Максимальный уровень!` |

#### Раздел помощи (`help_text`)

**Было:** 15 строк с развёрнутым описанием каждой кнопки.
**Стало:** 8 строк — кнопки чата свёрнуты в одну строку через `·`, команды собраны в одну строку.

## Исследование: CPA в нише Telegram dating-ботов

*Данные собраны из публичных отчётов: Business of Apps, Adjust, Telega.in, Sostav.ru, elama.ru, RichAds, Mapendo (2024–2025)*

---

### CPA по каналам привлечения

#### Telegram Ads (официальная реклама)

| Метрика | Значение |
|---------|----------|
| CPM (entertainment/dating) | €0.50–€3.00 |
| CTR | 0.9%–2.4% |
| CPA на старт бота (RU, оптимизировано) | €0.10–€0.50 |
| CPA на старт бота (среднее) | €0.35–€1.50 |
| Минимальный вход (через реселлера) | €2,250 |

#### Посевы в Telegram-каналах (RU рынок)

| Размер канала | Цена поста | CPA на старт |
|---------------|------------|--------------|
| Малый (1–10k подп.) | 500–3,000 ₽ | **15–60 ₽** |
| Средний (10–100k) | 3,000–30,000 ₽ | **40–200 ₽** |
| Крупный (100k+) | 15,000–100,000 ₽ | **75–500 ₽** |
| **Оптимум для старта** | **5,000–15,000 ₽** | **50–150 ₽** |

#### Яндекс Директ → бот

| Метрика | Значение |
|---------|----------|
| CPC (dating keywords) | 30–150 ₽ |
| Конверсия лендинг → старт | 15–40% |
| CPA на старт бота | **75–300 ₽** |

#### Instagram / TikTok (только EN/ES — в RU заблокированы)

### Окупаемость (Premium 299 ₽/мес, LTV 3 мес = 630 ₽)

| CPA канала | Break-even (1 мес) | Break-even (3 мес LTV) |
|-----------|---------------------|------------------------|
| 50 ₽ | 0.24% | 0.08% |
| 100 ₽ | 0.48% | 0.16% |
| 200 ₽ | 0.95% | 0.32% |
| 500 ₽ | 2.38% | 0.79% |

**Вывод:** При посевном CPA 100 ₽ и конверсии в Premium 1%+ канал окупается в первый месяц. Индустриальный бенчмарк dating apps — 7% платящих пользователей. Даже при 1–2% проект рентабелен.
